FormSubmit Activation Form

Используйте эту форму для активации сервиса FormSubmit для адреса support@dr-it.studio. После отправки формы проверьте почту на этом адресе для подтверждения.

Что произойдет дальше:

  1. После отправки этой формы, на адрес support@dr-it.studio придет письмо от FormSubmit.
  2. В этом письме будет ссылка для активации сервиса.
  3. Перейдите по этой ссылке для подтверждения адреса электронной почты.
  4. После активации все формы на вашем сайте, использующие FormSubmit с этим адресом, будут работать правильно.
