Политика конфиденциальности

Данная страница содержит специальную версию политики конфиденциальности для пользователей Forest Calculator, скачивающих приложение через Huawei AppGallery.

1. Какие данные мы собираем

  • Мы не собираем персональные данные автоматически.
  • Электронная почта может быть предоставлена только при обращении в поддержку.
  • Технические данные (тип устройства, версия ОС, язык интерфейса, частота использования, отчёты о сбоях) используются только для диагностики и улучшения сервиса.

2. Разрешения приложения

  • Доступ к памяти устройства — для сохранения и открытия файлов (PDF, Excel).
  • Доступ к интернету — для обновлений, рекламы и отправки писем.
  • Экспорт данных — для отправки расчётов через другие приложения.

3. Реклама и сторонние сервисы

  • Forest Calculator может показывать рекламу через партнёров (например, Huawei Ads).
  • Пользователь может выбрать тип рекламы при первом запуске и изменить в настройках.
  • Никакие персональные данные не передаются рекламным партнёрам.

4. Безопасность

  • Все расчёты и документы хранятся только на вашем устройстве.
  • Данные не передаются на серверы без вашего согласия.
  • Рекомендуем использовать блокировку экрана и другие меры защиты.

  • support@dr-it.studio

6. Согласие пользователя

Используя Forest Calculator через Huawei AppGallery, вы соглашаетесь с данной политикой конфиденциальности.
